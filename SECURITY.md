# Security Policy

OfflinePDFImageConverter is a local desktop application. It does not require
network access for PDF or image processing.

## Reporting a Vulnerability

If you find a security issue, please open a private security advisory on
GitHub if available, or contact the repository maintainer through the
repository's published contact method.

Please include:

- Affected version
- Steps to reproduce
- Impact
- Sample file only if it does not contain sensitive information

## Scope

Security issues in the application code and packaging are in scope.
Vulnerabilities in third-party libraries should also be reported to the
upstream project.

## Notes

The Windows executable may be unsigned unless a release explicitly states
otherwise. Verify release checksums before running downloaded binaries.
