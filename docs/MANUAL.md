# 使い方マニュアル

## 起動

`OfflinePDFImageConverter.exe` をダブルクリックします。インターネット接続は不要です。

## PDFを画像に変換

1. 画面上部の `PDF → 画像` を選びます。
2. `追加` でPDFを選ぶか、PDFファイルを画面へドラッグ＆ドロップします。
3. 画像形式を `PNG` または `JPEG` から選びます。
4. 解像度を `150 dpi`、`200 dpi`、`300 dpi`、`400 dpi`、`600 dpi` から選びます。
5. `出力先フォルダ` を選びます。
6. `変換開始` を押します。

出力名の例:

```text
sample_page001.png
sample_page002.png
```

同名ファイルがある場合は、上書きを避けるため末尾に番号を付けます。

## 画像をPDFに変換

1. 画面上部の `画像 → PDF` を選びます。
2. `追加` でJPEG/PNG画像を選ぶか、画像ファイルを画面へドラッグ＆ドロップします。
3. 必要に応じて `上へ` / `下へ` で順番を調整します。
4. ページサイズを選びます。
   - `A4縦`
   - `A4横`
   - `画像サイズに合わせる`
5. 余白を付ける場合は `余白あり` をオンにします。
6. `出力PDF` で保存先とファイル名を指定します。
7. `変換開始` を押します。

## PDF操作

画面上部の `PDF操作` を選びます。

PDF操作画面では、追加したPDFのページプレビューが表示されます。結合前にページの順番を確認したり、ページ削除時に消したいページを見ながら選べます。

ページプレビュー右上の表示切り替えで、`アイコン` と `リスト` を選べます。

- `アイコン`: ページの見た目を小さなプレビューで確認したいときに使います。
- `リスト`: ページ数やファイル名を素早く確認したいときに使います。

### 複数PDFを結合

1. `操作` で `複数PDFを結合` を選びます。
2. `追加` でPDFを複数選ぶか、PDFファイルを画面へドラッグ＆ドロップします。
3. 必要に応じて `上へ` / `下へ` で結合順を調整します。
4. ページプレビューで結合後のページ順を確認します。
5. `結合後のPDF` で保存先とファイル名を指定します。
6. `実行` を押します。

### 1ページずつ分割

1. `操作` で `1ページずつ分割` を選びます。
2. 分割したいPDFを選びます。複数PDFをまとめて処理できます。
3. ページプレビューで分割対象のページを確認します。
4. `出力先フォルダ` を選びます。
5. `実行` を押します。

出力名の例:

```text
sample_page001.pdf
sample_page002.pdf
```

### 選択ページを削除

1. `操作` で `選択ページを削除` を選びます。
2. PDFを1つだけ選びます。
3. ページプレビューで削除したいページにチェックを入れます。
4. `削除するページ` にページ番号が自動入力されます。直接入力もできます。
5. `出力PDF` で保存先とファイル名を指定します。
6. `実行` を押します。

入力例:

```text
1,3,5-7
```

元のPDFは上書きせず、指定ページを除いた新しいPDFを作成します。

## エラーが出た場合

- ファイルを他のアプリで開いている場合は閉じてください。
- 保存先に書き込みできるか確認してください。
- PDFがパスワード付き、破損、特殊な形式の場合は変換できないことがあります。
- 単体exeが起動しない場合は、一時フォルダへの書き込み制限がないか確認してください。
